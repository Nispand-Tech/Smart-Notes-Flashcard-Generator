# Smart Notes Flashcard Generator

A beginner-friendly Python + NLP project that converts study notes into flashcards.

## Features

- Read notes from a text file
- Sentence tokenization using NLTK
- Basic text cleaning
- Automatic flashcard generation

## Technologies Used

- Python
- NLTK
- Git
- GitHub
